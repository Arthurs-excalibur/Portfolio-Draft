# 🏯 Samurai Champloo Portfolio

A high-fidelity, interactive portfolio website themed after the legendary anime *Samurai Champloo*. This project blends modern web development with a lo-fi hip-hop aesthetic, featuring manga-inspired transitions, dynamic animations, and a curated sensory experience.

## 📸 Preview

![Samurai Champloo Portfolio Preview](./public/preview.png)

---

## ✨ Features

- **Dynamic Hero Section**: A cinematic entrance featuring stylized silhouettes and iconic imagery.
- **Manga-Style Panels**: Interactive sections designed to feel like moving through a manga volume.
- **Lo-Fi Aesthetic**: Custom cursors, film grain textures, and a sun-faded parchment color palette.
- **Background Music Player**: Fully integrated audio player featuring curated lo-fi tracks.
- **Fully Responsive**: Optimized for every device, from mobile scrolls to desktop blades.
- **Smooth Animations**: Powered by Framer Motion for a premium, lightweight feel.

## 🛠️ Tech Stack

- **Framework**: [Next.js 15](https://nextjs.org/) (App Router)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Typography**: Custom Google Fonts (Outfit, Inter)
- **Deployment**: Optimized for [Vercel](https://vercel.com/)

## 🚀 Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## 📜 Layout Structure

- `/src/sections`: Contains the main page sections (Hero, About, Projects, etc.).
- `/src/components`: Reusable UI components like the Manga Panels and Custom Cursor.
- `/public`: Media assets, including the lo-fi soundtrack and preview images.
